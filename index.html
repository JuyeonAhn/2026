배너 제작 가이드

위 미리보기는 배너 제작 툴입니다. 파일 또는 링크를 넣고 적용하기를 누르면 미리보기에 바로 반영되며, 이미지를 바꿔가며 여러 번 테스트할 수 있습니다. 텍스트·컬러를 정한 뒤 마지막에 미리보기에 반영으로 확정하세요.

1

이미지 첨부

배너로 사용할 이미지를 파일로 선택하거나, 이미지 주소(URL)를 입력한 뒤 각각 적용하기를 눌러 미리보기에 반영하세요. 여러 번 바꿔가며 테스트할 수 있습니다.

파일 선택 또는 여기에 이미지를 끌어다 놓으세요

이미지 검색

검색어를 입력하면 무료 이미지를 찾아 배너에 적용할 수 있습니다. Pexels는 가입 즉시 API 키 발급(심사 없음)으로 바로 테스트할 수 있습니다.

2

컬러 팔레트

이미지 반영 시 추출된 기본 주조색을 확인하고, 필요 시 변경한 뒤 기본과 비교할 수 있습니다.

기본 주조색 (이미지에서 추출) 이미지 반영 후 표시
주조색 변경하기
3

타이틀·서브 입력 (필수)

미리보기에 함께 표시할 텍스트를 입력하세요.

4

미리보기에 반영

첨부한 이미지(와 입력한 텍스트·주조색)를 상단 배너의 첫 번째 슬라이드에 적용합니다.

반영 후 상단 배너에서 노출 형태를 확인하세요. 창 크기를 조절해 반응형을 확인할 수 있습니다.

적용 규칙 (참고)

아래 규칙은 위 미리보기에 자동 적용됩니다. 제작 시 준수할 사항입니다.

브레이크포인트 및 크기
  • 1194px 이상: 배너 높이 300px 고정. 타이틀 32px, 서브 24px. 텍스트 좌측 정렬, 좌측 패딩 120px.
  • 769px ~ 1193px: 높이 260px. 타이틀 22px, 서브 15px. 좌측 정렬. 좌우 이동 버튼과 겹치지 않도록 패딩 88px.
  • 768px 이하: 이미지 영역(상단)과 텍스트 영역(하단) 분리. 타이틀 20px, 서브 14px. 텍스트 가운데 정렬.
  • 480px 이하: 이미지 높이 200px. 타이틀 18px, 서브 13px.
텍스트 제한
  • 타이틀: 30자 제한 (초과 시 말줄임 … 처리)
  • 서브 카피: 50자 제한 (초과 시 말줄임 … 처리)
배경·딤드 그라데이션
  • 각 슬라이드 이미지에서 주조색을 추출해 배경색으로 사용. 동일 이미지면 항상 같은 색이 나오도록 안정화.
  • 1194px 이상 / 769~1193px: 딤드 그라데이션은 화면 너비의 좌측 50%만 적용. 좌(높은 오퍼시티) → 중간(투명)으로 이어짐.
  • 768px 이하: 이미지 하단 1/3만 딤드 적용. 상단 2/3는 이미지 그대로 노출, 텍스트 영역과 색으로 자연스럽게 연결.
슬라이드 동작
  • 무한 루핑: 마지막 장 → 다음 버튼 시 첫 장으로, 첫 장 → 이전 버튼 시 마지막 장으로 끊김 없이 이어짐.
  • 좌우 이동 버튼과 하단 인디케이터로 이동 가능. 인디케이터는 현재 슬라이드 번호와 동기화됩니다.

배너 파일을 프로젝트 폴더에 넣고, 위 #banner-config에서 이미지 파일명·타이틀·서브를 수정하면 미리보기에 반영됩니다.
