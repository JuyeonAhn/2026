환상의 밤, 빛나는 숲속 이야기

몽환적인 발광 식물과 신비로운 생명들이 당신을 기다립니다.

배너 제작 가이드

위 미리보기는 업로드한 이미지·텍스트가 실제 노출되는 형태입니다. 아래 규칙을 준수해 제작해 주세요.

파일명 규칙 (레이블)
  • 1장만 사용: banner.jpg (또는 banner_1.jpg)
  • 여러 장 사용: banner_1.jpg, banner_2.jpg, banner_3.jpg … 순서대로 번호 부여
  • 이미지와 함께 사용할 타이틀·서브 카피는 이 페이지 상단의 #banner-config 설정에서 수정할 수 있습니다.
브레이크포인트 및 크기
  • 1194px 이상: 배너 높이 300px 고정. 타이틀 32px, 서브 24px. 텍스트 좌측 정렬, 좌측 패딩 120px.
  • 769px ~ 1193px: 높이 260px. 타이틀 22px, 서브 15px. 좌측 정렬. 좌우 이동 버튼과 겹치지 않도록 패딩 88px.
  • 768px 이하: 이미지 영역(상단)과 텍스트 영역(하단) 분리. 타이틀 20px, 서브 14px. 텍스트 가운데 정렬.
  • 480px 이하: 이미지 높이 200px. 타이틀 18px, 서브 13px.
텍스트 제한
  • 타이틀: 30자 제한 (초과 시 말줄임 … 처리)
  • 서브 카피: 50자 제한 (초과 시 말줄임 … 처리)
배경·딤드 그라데이션
  • 각 슬라이드 이미지에서 주조색을 추출해 배경색으로 사용합니다. 동일 이미지면 항상 같은 색이 나오도록 안정화되어 있습니다.
  • 1194px 이상 / 769~1193px: 딤드 그라데이션은 화면 너비의 좌측 50%만 적용. 좌(높은 오퍼시티) → 중간(투명)으로 자연스럽게 이어짐.
  • 768px 이하: 이미지 하단 1/3만 딤드 그라데이션 적용. 상단 2/3는 이미지가 그대로 노출되며, 텍스트 영역과 색으로 자연스럽게 연결.
슬라이드 동작
  • 무한 루핑: 마지막 장에서 다음 버튼 → 첫 장으로, 첫 장에서 이전 버튼 → 마지막 장으로 끊김 없이 이어짐.
  • 좌우 이동 버튼과 하단 인디케이터로 이동 가능. 인디케이터는 현재 보고 있는 실제 슬라이드 번호와 동기화됩니다.

배너 파일을 프로젝트 폴더에 넣고, 위 #banner-config에서 이미지 파일명·타이틀·서브를 수정하면 미리보기에 반영됩니다.
